# board

* 구동 방법
    1. git clone https://github.com/dew2jy/board.git
    2. board 디렉토리로 이동
    3. mvn package
    4. target 디렉토리로 이동
    5. java -jar board-0.0.1-SNAPSHOT.jar
    6. 브라우저에서 http://localhost:8080/ 접속
    7. 기본설정 계정인 username : user@email.com / password : user 입력 후 로그인
